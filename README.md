# Welcome to Obsidian

这是我的 Obsidian 知识库，按照 PARA 方法组织：

| 文件夹 | 用途 |
|--------|------|
| `00-收件箱/` | 临时笔记，快速记录，定期整理 |
| `10-日记/` | 每日日志，复盘思考 |
| `20-项目/` | 进行中的项目（1688 店铺运营、开发等） |
| `30-知识库/` | 已整理的知识、学习笔记 |
| `40-归档/` | 已完成/不再活跃的内容 |

常用快捷键：
- `Cmd+N` 新建笔记
- `Cmd+O` 快速打开
- `Cmd+P` 命令面板
- `Cmd+E` 切换编辑/预览
- `[[` 创建内部链接
