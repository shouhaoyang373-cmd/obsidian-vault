# 1688 店铺诊断 - 操作流程

## 快捷指令

双击桌面文件夹里的脚本即可自动生成诊断报告：

```
~/Desktop/1688店铺分析/店铺诊断.command
```

**前置条件**：Chrome 保持登录 1688 卖家工作台。

---

## 数据采集流程

### 第一步：提取 Chrome Cookies

```python
import browser_cookie3
cj = browser_cookie3.chrome(domain_name='1688.com')
cookies = [f'{c.name}={c.value}' for c in cj if c.value]
```

### 第二步：抓取卖家工作台数据

目标页面：

| 页面 | 路径 | 数据 |
|------|------|------|
| 首页 | `#/` | 店铺概览 / 订单 / GMV |
| 服务体验 | `_path_=sellerPro/lvyue/new-lighthouse-ai` | 新灯塔四维得分 |
| 成长中心 | `_path_=sellerPro/seller-center/seller-growth` | 店铺等级 / 诚信通 |

### 第三步：抓取生意参谋数据

| 页面 | URL | 数据 |
|------|-----|------|
| 核心看板（日） | `sycm.1688.com/ms/home/home?dateType=day` | 展现/访客/支付 |
| 核心看板（周） | `sycm.1688.com/ms/home/home?dateType=week` | 周趋势 |
| 核心看板（月） | `sycm.1688.com/ms/home/home?dateType=month` | 月趋势 |
| 入店关键词（7天） | `sycm.1688.com/ms/flow/keywordAnalysis` 点「近7天」 | 展现+排名 |
| 入店关键词（30天） | `sycm.1688.com/ms/flow/keywordAnalysis` 点「近30天」 | 展现+排名+对比 |

### 第四步：MTOP 签名（进阶）

```
sign = MD5(token & 时间戳 & "12574478" & data)
token = _m_h5_tk cookie 中 '_' 前半段
```

来源：[ihmily/1688-Decryptor](https://github.com/ihmily/1688-Decryptor)

### 第五步：生成 PDF 报告

Playwright headless 渲染 HTML → A4 PDF。

1. 店铺概览

8. 竞店对比
9. 流量渠道诊断
10. 产品货品结构
11. 转化 & 询盘
12. 竞品 & 市场对标
13. 定制类目专项

## MTOP 签名公式

```
sign = MD5(token & t & appKey & data)

token   = _m_h5_tk 前半段
t       = 13位毫秒时间戳
appKey  = 12574478（固定值）
data    = 请求 JSON
```

请求时必须 Cookie 中包含相同的 `_m_h5_tk`，否则鉴权失败。
